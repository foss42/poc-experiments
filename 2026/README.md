# 2026 GSoC PoCs

This directory contains candidate Proof of Concept projects for GSoC 2026.

## Available PoCs

- `open_responses_explorer/`: Open Responses + GenUI exploration platform.

## Notes for Reviewers

- Each PoC folder should be self-contained.
- Prefer source code, tests, and docs only.
- Avoid committing generated build artifacts or platform scaffolding unless explicitly required.
